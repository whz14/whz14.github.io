# 约定用语
### 层级
层级从高到低以此为**部分**（如跑步学习笔记），**章**（如跑步基础知识），**节**（如运动中的物理学），每一节占一个页面

一**部分**中可能会有单个或多个章，章中，节中同理，如本节就是本部分中唯一的章唯一的节
